# 视频切片

## 1.问题分析

除了单纯的偏爱以外，如何嗅探一个主播是否具备剪出良好切片的问题应该是一个学问，首先需要了解行情吧？不过需要了解的问题太多，建议先找一两个国内的主播来进行分析，以及一两个做切片的 `man` 做分析。具体需要研究下面的一些问题：

- 现在有哪些头部、中部的虚拟主播？
- 这种主播的粉丝量为？
- 切片发布数量？
- 同行切片 `man` 的更新频率？
- 确认对标主播和对标切片 `man` 是谁？

## 2.工具准备

1. 视频获取：`Bilibili` 回放并且使用 [Snapany](https://snapany.com/zh/bilibili) 来获取 或 `OBS` 录制
2. 音频分离：使用 [Lalal](https://www.lalal.ai/zh-hans/) 来快速进行人声分离，或者安装 [Ultimate Vocal Remover GUI](https://github.com/Anjok07/ultimatevocalremovergui#ultimate-vocal-remover-gui-v56) 来本地分离
3. 素材整理：整理视频中可能出现的所有素材，渠道如下
   - [いらすとや](https://www.irasutoya.com/)
   - [Telegram Emoji](https://emojipedia.org/telegram) 和 [Microsoft Emoji](https://emojipedia.org/zh/microsoft/windows-11-23h2-june-2024-update)
4. 动画制作：`Adobe After Effects`，先基本做个视频编排，然后做添加素材做基本动画
5. 风格统一：需要有一些自己的个人风格，全屏水印（零式演算&虚拟主播）
6. 视频编排：`CapCut`
