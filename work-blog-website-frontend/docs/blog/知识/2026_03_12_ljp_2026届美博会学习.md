## 2.核心正文

### 2.1.商家分类

| 角色            | 含义                                                         |
| --------------- | ------------------------------------------------------------ |
| **合作人**      | 品牌合作人是 **与品牌方轻资产合伙** 的合作者。以资源、资金或场地入股，不买断货，按业绩与品牌分成，兼具 B 端铺店与 C 端服务能力。 |
| **经销商**      | 经销商是 **从工厂直接买断货权** 的商家。他们像一级批发商，自己出钱把产品货物买下，拥有这些货的所有权。然后把货批发给下级商家（如美容院、门店等 `B` 端客户），需要自己自负盈亏。 |
| **代理商**      | 代理商是 **获得工厂授权** 的商家。他们不买断货权，而是代表工厂在特定区域或渠道内进行销售。他们更像工厂的 “区域代言人” 或 “销售分公司”，通常赚取佣金。 |
| **渠道商**      | 渠道商是 **有人脉、有渠道资源** 的商家。他们不局限于一个品牌，主要帮工厂 / 品牌把货铺进各类门店或渠道，赚取通路费、进场费或差价。 |
| **加盟商**      | 加盟商是 **付费加入品牌体系** 的经营者。缴纳加盟费与保证金，使用品牌全套模式开店，直接面向终端顾客赚门店利润。 |
| **零售商**      | 零售商是 **直接面向普通顾客** 的终端商家。从经销商 / 代理商 / 渠道商拿货，自主选品定价，直接卖给最终消费者。 |
| **分销商**      | 分销商是 **层级分销中的下级流通商**。从代理或经销商处拿货，再向下级小商家或门店分销，以小差价走量为主。 |
| **代购 / 带货** | 代购 / 带货是 **靠流量推荐销售** 的推广者。不开店或不囤货，通过直播 / 私域等渠道推荐成交，赚取佣金或分成。 |

### 2.4.成分解读

#### 2.4.1.关于精华水和精粹水的区别

一般来说精粹水回避精华水更加浓缩一些，但是基本上看厂家而定。

#### 2.4.2.关于肉毒杆菌和肉毒素的区别

| 名称    | 是什么 |有无活菌| 安全性      |
|-------|-----|--|----------|
| 肉毒杆菌  |细菌| 	有活菌 | 污染食物会中毒  |
| 肉毒素   |细菌分泌的毒素蛋白| 完全灭活，无活菌 | 正规注射比较安全 |

#### 2.4.3.舒缓剂成分

`SymSitive®1609` 是德国 **巴斯夫（`BASF`）** 旗下的经典舒缓抗刺激专利成分，全称是 `SymSitive® 1609 (SymCalmin®)`，化学名：四氢甲基嘧啶羧酸（依克多因）+ 羟苯基丙酰胺苯甲酸（`HPPB`）。

- 作用原理：它能直接抑制 `TRPV1` 受体（这是皮肤感知疼痛、灼热、刺痛的关键受体），从而快速缓解高浓度酸（如 30% 水杨酸）带来的刺痛、泛红和灼热感，同时还能辅助修护皮肤屏障
- 技术背景：宣传里提到的 “源自诺奖 `TRPV1` 发现”，是指 `TRPV1` 受体的发现曾获 `2021` 年诺贝尔生理学或医学奖，而 `SymSitive®1609` 正是基于这一机制开发的靶向舒缓成分

#### 2.4.3.关于精油和香精的区别

- 精油 = 植物本身，天然，疗愈，贵
- 香精 = 人工香味，好闻，便宜，量大

##### 2.4.4.关于面霜和面膜霜的区别

- **面霜（Day/Night Cream）**：是 **日常护肤的最后一步**，主打 **锁水、保湿、修护**，需要留在皮肤上，不需要清洗，是护肤的 “封闭锁层”。
- **面膜霜（Cream Mask / Wash-off Mask）**：本质是 **膏状 / 霜状面膜**，属于 **周期性护理产品**，需要厚敷一段时间后洗掉（或擦拭掉），主打 **密集修护、清洁、焕肤**，不能当日常面霜用

### 2.6.品牌公司

- 修丽可（`SkinCeuticals`）：`1997` 年由美国杜克大学皮肤科学家 `Sheldon Pinnell` 教授创立，品牌名由「`Skin`（皮肤）+ `Pharmaceuticals`（制药）」组合而成，强调医学级科研背景，修丽可（`SkinCeuticals`） 在 `2017` 年 被欧莱雅集团（`L'Oréal`）全资收购。
- 科颜氏（`Kiehl's`）：欧莱雅集团旗下，面向普通消费者的高端护肤品牌
- 韵姿（`VAGHEGGI`）：意大利高端专业院线护肤品牌，`1975` 年创立，主打天然植萃 + 矿物元素 + 芳疗 `SPA`，定位高端功效型院线
- 尚美国际：专门给美容院供货、培训、做活动的 “美业服务商”

### 2.7.面部打版

**品牌方 / 美容院** 找一个人，当场在她脸上做项目（刷酸、祛斑、抗衰、仪器等）把前后对比当成活广告，给现场其他客户、经销商看：“你看！效果这么明显！”。
