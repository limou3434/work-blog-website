# 五坂库

- **出场**：该虚拟生命又 `Docker` 容器来进行部署，同时采用 `K8s` 作为组合，因此开场动画就是各个身体部位的 `Docker` 开始做运动，然后通过 `K8s` 来进行一个组合，类似魔法阵一样出现，同时有个数据流上下流动的展示，并且还有对应的启动指令动画，以及关机动画
- **色调**：主色调以 `Ubuntu` 深紫为基底，搭配浅紫、蓝粉，点缀冷白代码光、`Y2K CD` 的镭射银，还有类似机铠种的高对比的橙色瞳孔
- **眼睛**：左眼为淡紫色瞳孔，瞳孔中心有的 `Ubuntu 24.04` 的皇冠 `Logo`。右眼白处有细微的淡紫色代码纹路，这里会显示系统后台运行的日志提示，平时情绪平稳时纹路不明显若隐若现，在频繁调动力量时彻底亮起，并且已非常快的速度进行滚动
- **装饰**：
  - **耳机**：紫色耳机戴在脖子上，并且有霓虹灯特效，会播放舞萌视频
  - **专辑**：采用 `Y2K` 经典的透明塑料 `CD` 盒，专辑在播放的时候会闪过调用代码，最后跳出播放音乐的立体条状图
- **衣服**：
  - 上身为对称露肩针织短款劲辣上衣（参考模型为银狼），衣身织纹暗藏 `Ubuntu Logo` 的线条化纹样，领口与下摆边缘有“数据消融”效果——布料逐渐过渡为透明代码流，仿佛随时会融入虚拟空间。衣料自带细闪光泽，光线照射下会浮现细碎的 `01` 字符印记

  - 下身搭配小短裙，裙身采用哑光紫皮质面料，侧面有金属按扣装饰，扣面刻着 `Ubuntu` 常用指令“`apt`”，裙摆内侧印着淡紫色的开源协议条文，走动时裙摆扬起，条文会随动作发出微弱荧光。裙摆边缘有不规则毛边，毛边处呈像素化破碎感，呼应数据生命体的特质
  - 脚踩紫色过膝长筒袜，袜口有白色条纹，条纹上滚动着慢速代码；脚上是银色厚底乐福鞋，鞋头镶嵌透明水晶，水晶内封着微型 `Ubuntu` 系统启动界面的投影芯片。左手腕戴细款银色手链，链节处串联着迷你光盘吊坠（非核心道具，仅装饰），右手腕无配饰，便于动作时展现周身飘行的代码
- **体态**：
  - **身材**：少女偏向萝莉
  - **纹身**：左肩凸显纹身 `+/-0`，与初音原版一致，致敬初音未来，而且纹身表面有荧光涂层，在暗处或调动力量时会发出亮紫色光芒
- **性格**：孤独且好奇的，会不断学习各种没有见过的事物
